## 🚀 วิธีรันโปรเจกต์

### สิ่งที่ต้องมี

- [Node.js](https://nodejs.org/) เวอร์ชัน 18 ขึ้นไป
- npm (มาพร้อมกับ Node.js)

### ขั้นตอนการติดตั้งและรัน

```bash
# 1. ติดตั้ง dependencies
npm install

# 2. รันโปรเจกต์ (โหมด development)
npm run dev
```

เปิดเบราว์เซอร์ไปที่ **http://localhost:5173** 🎉

### คำสั่งอื่นๆ

| คำสั่ง | คำอธิบาย |
|-------|---------|
| `npm run dev` | รันแบบ development (มี hot reload) |
| `npm run build` | build สำหรับ production |
| `npm run preview` | ดูตัวอย่าง production build |
| `npm run lint` | ตรวจสอบโค้ดด้วย ESLint |

## 📱 วิธีเปิดบนมือถือ (เพื่อใช้กล้อง)

> ⚠️ เบราว์เซอร์บนมือถือ**ต้องใช้ HTTPS** ถึงจะเปิดกล้องได้

### วิธีที่ 1 — ทดสอบผ่าน LAN (ง่ายและเร็ว)

ติดตั้ง plugin สำหรับ HTTPS:

```bash
npm install -D @vitejs/plugin-basic-ssl
```

แก้ไฟล์ `vite.config.ts`:

```ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import basicSsl from '@vitejs/plugin-basic-ssl'

export default defineConfig({
  plugins: [react(), basicSsl()],
})
```

จากนั้นรัน:

```bash
npm run dev -- --host
```

เปิดมือถือ (ต้องต่อ WiFi เดียวกัน) แล้วเข้า `https://192.168.x.x:5173` → ยอมรับ certificate → ใช้กล้องได้!

### วิธีที่ 2 — Deploy ขึ้น Vercel (แนะนำสำหรับใช้งานจริง)

1. Push โค้ดขึ้น **GitHub**
2. ไปที่ [vercel.com](https://vercel.com) → Sign up ด้วย GitHub
3. กด **Import Project** → เลือก repo ของคุณ
4. Vercel จะ build + deploy อัตโนมัติ
5. ได้ URL `https://xxx.vercel.app` → เปิดในมือถือได้เลย ✅

## 🖼️ วิธีใส่รูปภาพของตัวเอง

### 📂 โครงสร้างโฟลเดอร์รูปภาพ

```
public/photos/
├── top.jpg      ← รูปบนยอดดาว (รูปหลัก)
├── 1.jpg        ← รูปบนต้นไม้
├── 2.jpg
├── 3.jpg
└── ...          ← ใส่ได้ไม่จำกัด
```

### ขั้นตอน

1. **เตรียมรูปภาพ** — แนะนำรูปสี่เหลี่ยมจัตุรัส หรือ 4:3 ขนาดไม่เกิน 500KB ต่อรูป
2. **วางรูปใน** `public/photos/` — ตั้งชื่อเป็น `top.jpg`, `1.jpg`, `2.jpg`, ... ตามลำดับ
3. **แก้จำนวนรูป** — ถ้าเพิ่ม/ลดรูป ให้เปิดไฟล์ `src/App.tsx` แล้วแก้บรรทัดนี้:

```ts
const TOTAL_NUMBERED_PHOTOS = 31; // ← เปลี่ยนตัวเลขนี้ตามจำนวนรูปที่มี
```

## 🖐️ วิธีควบคุมด้วยท่ามือ

> ระบบ AI จดจำท่ามือจะทำงานผ่านกล้อง — กดปุ่ม **DEBUG** มุมขวาล่างเพื่อดูภาพจากกล้อง

| ท่ามือ | การกระทำ | ผลลัพธ์ |
|--------|---------|---------|
| 🖐 **กางมือ** (Open Palm) | กระจาย (Disperse) | ต้นไม้ระเบิดกลายเป็นอนุภาคลอยฟุ้ง |
| ✊ **กำปั้น** (Closed Fist) | รวมตัว (Assemble) | อนุภาคทุกชิ้นรวมกลับเป็นต้นคริสต์มาส |
| 👋 **เลื่อนมือซ้าย-ขวา** | หมุนซ้าย-ขวา | เปลี่ยนมุมมองรอบต้นไม้ |
| 👋 **เลื่อนมือขึ้น-ลง** | ก้ม-เงย | เปลี่ยนมุมมองขึ้น-ลง |

## ⚙️ ปรับแต่งขั้นสูง

แก้ไขค่า `CONFIG` ในไฟล์ `src/App.tsx`:

```ts
const CONFIG = {
  colors: { ... },          // 🎨 เปลี่ยนสีต้นไม้ ไฟ กรอบรูป
  counts: {
    foliage: 15000,          // 🌿 จำนวนอนุภาคต้นไม้ (ลดลงถ้าเครื่องช้า)
    ornaments: 300,          // 📸 จำนวนรูปที่แขวน
    lights: 400              // 💡 จำนวนไฟกระพริบ
  },
  tree: { height: 22, radius: 9 }, // 📐 ขนาดต้นไม้
};
```
